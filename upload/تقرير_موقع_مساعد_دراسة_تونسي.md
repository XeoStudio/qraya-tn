## نظرة عامة على المشروع

### اسم المشروع
**مساعد دراسة تونسي** (Tunisian Study Assistant)

### الوصف
منصة تعليمية ذكية مخصصة للطلاب التونسيين، توفر مساعدة دراسية باستخدام الذكاء الاصطناعي. تساعد المنصة الطلاب في فهم الدروس، تلخيص المحتوى، إنشاء خطط دراسة، وحل الأسئلة التدريبية.

### الفئة المستهدفة
- طلاب التعليم الابتدائي
- طلاب التعليم الإعدادي
- طلاب التعليم الثانوي
- طلاب الباكالوريا

### اللغات المدعومة
- العربية (اللغة الأساسية)
- الفرنسية (للمحتوى التعليمي)
- الإنجليزية (للمحتوى التقني)

## الفكرة والأهداف

### الفكرة الأساسية
إنشاء مساعد ذكي شخصي لكل طالب تونسي يساعده في:
1. **شرح المفاهيم**: شرح مبسط ومفصل لأي مفهوم دراسي
2. **التلخيص**: تلخيص الدروس والمواد الدراسية
3. **خطط الدراسة**: إنشاء خطط دراسة مخصصة
4. **الأسئلة التدريبية**: توليد أسئلة MCQ مع التصحيح
5. **بطاقات المراجعة**: إنشاء بطاقات Q/A للمراجعة السريعة

### الأهداف الرئيسية
1. **تسهيل التعلم**: جعل التعلم أسهل وأكثر متعة
2. **التخصيص**: تكييف المحتوى حسب مستوى الطالب
3. **التفاعلية**: توفير تجربة تعليمية تفاعلية
4. **الوصول المجاني**: توفير محتوى مجاني أساسي
5. **الجودة**: محتوى عالي الجودة مخصص للمناهج التونسية

## قاعدة البيانات

### المخطط (Schema)

#### 1. User (المستخدم)
```prisma
model User {
  id            String    @id @default(cuid())
  email         String    @unique
  name          String?
  password      String
  avatar        String?
  
  // الملف الدراسي
  level         String?   // primary, preparatory, secondary, bac
  levelName     String?
  year          String?
  section       String?
  subjects      String?
  
  // الدور والحالة
  role          Role      @default(USER)
  status        UserStatus @default(ACTIVE)
  isVerified    Boolean   @default(false)
  
  // الاشتراك
  subscription  Subscription?
  usedPromoCodes PromoCodeUsage[]
  
  // Gamification
  points        Int       @default(0)
  streak        Int       @default(0)
  lastActive    DateTime?
  achievements  Achievement[]
  studyStats    StudyStat[]
  
  // النشاط
  sessions      Session[]
  activities    ActivityLog[]
  chats         Chat[]
  reports       ErrorReport[]
  
  // حقول الإدارة
  bannedAt      DateTime?
  bannedReason  String?
  notes         String?
  
  createdAt     DateTime  @default(now())
  updatedAt     DateTime  @updatedAt
}
```

#### 2. Subscription (الاشتراك)
```prisma
model Subscription {
  id            String        @id @default(cuid())
  userId        String        @unique
  user          User          @relation(...)
  
  plan          PlanType      @default(FREE)
  status        SubscriptionStatus @default(ACTIVE)
  
  startDate     DateTime      @default(now())
  endDate       DateTime?
  
  // صلاحيات الوصول
  agentMode     Boolean       @default(false)
  advancedAI    Boolean       @default(false)
  unlimitedChat Boolean       @default(false)
  priority      Boolean       @default(false)
  exportPDF     Boolean       @default(false)
  ocrUnlimited  Boolean       @default(false)
  customPlans   Boolean       @default(false)
  
  // حدود الاستخدام
  chatsUsed     Int           @default(0)
  ocrUsed       Int           @default(0)
  quotaLimit    Int           @default(50)
}
```

#### 3. PromoCode (كود التفعيل)
```prisma
model PromoCode {
  id            String    @id @default(cuid())
  code          String    @unique
  planType      PlanType
  
  duration      Int?      // أيام، null = مدى الحياة
  maxUses       Int?
  usedCount     Int       @default(0)
  
  // المميزات
  agentMode     Boolean   @default(false)
  advancedAI    Boolean   @default(false)
  unlimitedChat Boolean   @default(false)
  priority      Boolean   @default(false)
  exportPDF     Boolean   @default(false)
  ocrUnlimited  Boolean   @default(false)
  customPlans   Boolean   @default(false)
  
  isActive      Boolean   @default(true)
  createdBy     String?
  expiresAt     DateTime?
}
```

## نظام المستخدمين

### أنواع المستخدمين

#### 1. مستخدم عادي (USER)
```
الصلاحيات:
- استخدام الأدوات الأساسية
- 50 محادثة يومياً
- رفع الملفات
- الأمثلة المجانية

القيود:
- لا يمكن استخدام وضع المعلم الذكي
- لا يمكن تصدير PDF
- حدود على OCR
```

#### 2. مستخدم مميز (PREMIUM)
```
الصلاحيات:
- جميع مميزات المستخدم العادي
- وضع المعلم الذكي
- محادثات غير محدودة
- ذكاء اصطناعي متقدم
- أولوية في الرد
- تصدير PDF
- OCR غير محدود
```

#### 3. مدير النظام (ADMIN)
```
الصلاحيات:
- جميع مميزات PREMIUM
- لوحة تحكم الإدارة
- إدارة المستخدمين
- إنشاء أكواد التفعيل
- عرض السجلات والأخطاء
- حظر/إلغاء حظر المستخدمين
```

### نظام المصادقة

#### التسجيل
```javascript
// البيانات المطلوبة
{
  action: 'register',
  email: string,
  password: string,  // 6 أحرف على الأقل
  name: string,
  level: string,     // اختياري
  year: string,      // اختياري
  section: string    // اختياري
}
```

#### تسجيل الدخول
```javascript
// البيانات المطلوبة
{
  action: 'login',
  email: string,
  password: string
}

// الاستجابة
{
  success: true,
  user: {
    id, email, name, role, points, streak, ...
  },
  subscription: { plan, features, ... },
  token: "jwt_token"
}
```

#### الحماية
- Rate Limiting (5 محاولات ثم حظر 15 دقيقة)
- تشفير كلمات المرور بـ bcrypt (12 rounds)
- JWT للجلسات (صلاحية 7 أيام)
- تحقق من صحة البريد الإلكتروني

## واجهات API

### 1. المصادقة (`/api/auth`)

| الطريقة | الإجراء | الوصف |
|---------|---------|-------|
| POST | register | إنشاء حساب جديد |
| POST | login | تسجيل الدخول |
| GET | - | جلب بيانات المستخدم الحالي |

### 2. المحادثة (`/api/chat`)
```javascript
POST /api/chat
{
  message: string,
  context: string,    // المحتوى المرفوع
  history: [{ role, content }]
}

// الاستجابة
{ response: string }
```

### 3. الإدارة (`/api/admin`)
```javascript
// يجب أن يكون المستخدم ADMIN
GET /api/admin?action=stats     // إحصائيات
GET /api/admin?action=users     // قائمة المستخدمين
GET /api/admin?action=logs      // سجلات النشاط
GET /api/admin?action=promocodes // الأكواد

POST /api/admin
{
  action: 'create-promo',
  code: string,
  planType: string,
  duration: number,
  maxUses: number
}

POST /api/admin
{
  action: 'ban-user',
  userId: string,
  reason: string
}
```

### 4. المستخدم (`/api/user`)
```javascript
POST /api/user
{
  action: 'update-profile',
  name: string
}
```

### 5. الاشتراك (`/api/subscription`)
```javascript
POST /api/subscription
{
  code: string  // كود التفعيل
}
```

### 6. التلخيص (`/api/summarize`)
```javascript
POST /api/summarize
{
  content: string,
  level: string
}
```

### 7. الأسئلة (`/api/quiz`)
```javascript
POST /api/quiz
{
  topic: string,
  count: number,
  difficulty: string
}
```
